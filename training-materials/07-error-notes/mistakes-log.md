# Mistakes Log

SQL 연습 중 막힌 명령, 오류 메시지, 원인, 해결 방법을 기록하는 오답 노트이다.

평가 직전에는 같은 실수를 반복하지 않도록 최근 기록부터 다시 확인한다.

---

## 기록 방법

오류가 발생하면 아래 항목을 남긴다.

```text
[ ] 실행한 명령
[ ] 발생한 오류 메시지
[ ] 원인 추정
[ ] 해결 방법
[ ] 다음에 조심할 점
```
